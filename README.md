# KEJ Calendar

KEJ Calendar is a lightweight, browser-based calendar application built as a single-page web project.

## Live Demo

Try it here: **https://kejubayer.github.io/kej-calendar**

## Project Details

- **Type:** Static web app
- **Main file:** `index.html`
- **Purpose:** Provide a simple, fast calendar interface that runs directly in the browser.

## How to Use

1. Open the live demo link above, or
2. Clone this repository and open `index.html` in your browser.

## Local Development

```bash
git clone https://github.com/kejubayer/kej-calendar.git
cd kej-calendar
# then open index.html in your browser
```

## Deployment

This project is deployed with GitHub Pages.
